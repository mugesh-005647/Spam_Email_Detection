# 📧 Spam Email Detection using Machine Learning

A web application that predicts whether an email or SMS is **Spam** or **Not Spam** using Machine Learning.

## 🚀 Features

- Spam and Ham classification
- Machine Learning model using Multinomial Naive Bayes
- CountVectorizer for text feature extraction
- Prediction confidence
- User-friendly Streamlit interface
- Real-time prediction

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle

## ▶️ Installation

Clone the repository.

```bash
git clone https://github.com/mugesh-005647/Spam_Email_Detection.git
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Run the application.

```bash
streamlit run app.py
```
